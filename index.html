What Is ConnectionSpotter?

ConnectionSpotter is a Firefox add-on that enhances the news reading experience by revealing hidden connections between politicians, business people, corporations, and organizations. ConnectionSpotter builds on Poligraft, OpenSecrets, and LittleSis.

Try ConnectionSpotter out on these articles:

  • F.C.C. Seeks a New Path on 'Net Neutrality' Rules
  • Report Opens Way to Approval for Keystone Pipeline
  • Profit Increases, but Revenue Falls at Northrop

Installation Instructions

  1. Download the add-on by clicking on the .zip link above.
  2. Uncompress/unzip the zip file. It contains a single .xpi file.
  3. Download Firefox here if you don't already have it.
  4. Open Firefox and press Cmd-o (or Ctrl-o if on windows). Navigate to the .xpi file you just downloaded and click Open. Click "Install Now" when prompted.

IMPORTANT: ConnectionSpotter may not work if you are not using the latest version of Firefox. ConnectionSpotter requires version 26 or later. At time of publishing, if you are using the latest ESR release, then your browser does not support ConnectionSpotter. Download a standard distribution before continuing. You can do so with the link in step 3.

CARLETON: Lab machines at Carleton do not have the latest version of Firefox, so download a newer version from the link provided above. There is also a conflict between ConnectionSpotter and one of the add-ons that is installed on the lab machines. You must reset the Firefox profile before performing step 4. This link explains how to do that (it's easy!).

If you ever want to uninstall ConnectionSpotter, follow the instructions here.

How To Use

After installing, a small eye icon will appear in the add-on bar. This icon will usually be located in the lower right corner of the browser window. If you don't see the icon, your add-on bar may be hidden. Unhide the add-on bar by following these instructions.

When you are reading an article, click the eye to find connections!

The ConnectionSpotter sidebar can be scrolled up and down. You may also resize it by dragging its edge for easier viewing.
